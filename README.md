# QSAR_ADMET_Prediction

This repository demonstrates QSAR-based ADMET prediction using
cheminformatics and machine learning techniques.

## What this project covers
- Molecular descriptor generation using RDKit
- Dataset curation from public sources (ChEMBL, toxicity datasets)
- QSAR model building for ADMET properties
- Machine learning models (Random Forest, LightGBM, XGBoost)
- Model evaluation and validation

## Applications
- Early-stage drug discovery
- ADMET risk assessment
- Lead optimization support

## Tools & Technologies
- Python
- RDKit
- scikit-learn
- LightGBM / XGBoost
- Pandas, NumPy
